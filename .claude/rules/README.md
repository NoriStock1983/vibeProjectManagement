# .claude/rules フォルダについて

このフォルダは「Claude に守ってほしいコーディングルール・作業規約」を記載する場所です。
システムの非機能要件ではなく、Claude がコードを書く・変更する際に従うべき指示・制約を管理します。

## ルールの種類

- **無条件ルール**（frontmatter なし）: 常時読み込まれ全ファイルに適用
- **パス限定ルール**（`paths` frontmatter あり）: 対象ファイル操作時のみ読み込み

## 主な記載内容

コーディング規約、テスト方針、セキュリティ要件、Git ワークフロー、設計パターンなど。

## CLAUDE.md との使い分け

- `CLAUDE.md`: プロジェクト全体の概要・主要な指示
- `.claude/rules/`: テーマごとに分割した詳細ルール
