## 1. 前提条件ログ

前提条件とは、「真と仮定して計画を進めているが、検証が完了していない事実」のこと。

| A-003 | リソース | | | | | 未確認 |
| A-004 | 外部依存 | | | | | 未確認 |

**ステータスの凡例：**
- **未確認**：前提として設定したが、まだ検証されていない
- **確認済**：前提が正しいことが確認された
- **要更新**：前提が変化した・または崩れた可能性がある
